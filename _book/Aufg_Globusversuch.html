Nutzen Sie die Gittermethode gegliedert in 10 Hypothesen, um herauszufinden, welcher der wahrscheinlichste Wasseranteil des Globus ist, wenn wir bei 20 Durchführungen 14-mal Wasser getroffen haben.

set.seed(42)
d <-
  tibble(
    # definiere das Gitter: 
    p_Gitter = seq(from = 0, to = 1, length.out = 10),
    # bestimme den Priori-Wert:       
    Priori  = 1) %>%  
    mutate(
      # berechne Likelihood für jeden Gitterwert:
      Likelihood = dbinom(14, size = 20, prob = p_Gitter),
      # berechen unstand. Posteriori-Werte:
      unstd_Post = Likelihood * Priori,
      # berechne stand. Posteriori-Werte (summiert zu 1):
      Post = unstd_Post / sum(unstd_Post))  
d

Verwenden Sie die Gittermethode für den exakt gleichen Versuch noch einmal, aber verwenden Sie nun zur Errechnung der Posteriori-Wahrscheinlichkeit das Bayestheorem in R.

set.seed(42)

d2 <-
  tibble(
    # definiere das Gitter: 
    p_Gitter = seq(from = 0, to = 1, length.out = 10),
    # bestimme den Priori-Wert:       
    Priori  = 1) %>%  
    mutate(
      # berechne Likelihood für jeden Gitterwert:
      Likelihood = dbinom(14, size = 20, prob = p_Gitter),
      # berechne stand. Posteriori-Werte über das Bayestheorem:
      Posteriori = Likelihood * Priori / mean(Likelihood)) #Likelihood * Priori bildet den Teil über dem Bruchstrich ab
                                                           #die Evidenz errechnet sich als Mittelwert aller Likelihoods

Nehmen Sie an, Sie könnten begründen, dass ein Wasseranteil unter 50 % nicht existiert. Formen Sie die Gittermethode so, dass Werte unterhalb von 50 % Wasseranteil iene Posteriori-Wahrscheinlichkeit von 0 bekommen. Tipp: Nutzen Sie die Funktion case_when(), um den Prior zu definieren.

set.seed(42)

d3 <-
  tibble(
    # definiere das Gitter: 
    p_Gitter = seq(from = 0, to = 1, length.out = 10),
    # bestimme den Priori-Wert:       
    Priori  = case_when(p_Gitter < 0.5 ~ 0, #alle p_Gitter-Werte unter 0.5 werden mit 0 verrechnet
                        p_Gitter >= 0.5 ~ 1)) %>%  #alle p_Gitter-Werte über oder gleich 0.5 sind gleichwahrscheinlich und werden mit 1 verrechnet
    mutate(
      # berechne Likelihood für jeden Gitterwert:
      Likelihood = dbinom(14, size = 20, prob = p_Gitter),
      # berechne stand. Posteriori-Werte über das Bayestheorem:
      Posteriori = Likelihood * Priori / mean(Likelihood)*0.1)
d3
d3%>%
  filter(p_Gitter >0.5)%>%
  summarise(sum(Posteriori))

Nutzen Sie die Funktion dbinom(), um zu ermitteln, wie groß die Wahrscheinlichkeit ist, das wir nur eines von fünf Malen treffen, obwohl die Trefferwahrscheinlichkeit bei 50 % liegen.

dbinom(1, size = 5, #Wie oft erhalten wir einen von fünf Treffern, wenn die Chance zu treffen 50/50 steht?
       prob = .5)
[1] 0.15625

Unser Szenario, also dass wir eines von fünf Malen treffen, hat eine Eintrittswahrscheinlichkeit von ca. 16 Prozent, wenn die Trefferchance tatsächlich 50/50 steht. Aber wenn die Wahrscheinlichkeit so gering ist, dann kann es doch nicht sein, dass uns dieses Szenario trotzdem so regelmäßig passiert?

Berechnen wir ein Mal die Wahrscheinlichkeit eines von fünf Malen zu treffen, wenn die Trefferahrscheinlichkeit bei 20 Prozent läge.

dbinom(1, size = 5, #Wie oft erhalten wir einen von fünf Treffern mit einer Trefferwahrscheinlichkeit von 20 % ?
       prob = .2) # Trefferwahrscheinlichkeit liegt bei 20 %
[1] 0.4096

Das Ergebnis sagt uns: Wenn die Trefferwahrscheinlichkeit bei 20 Prozent liegt, dann sollten wir in ca. 41 Prozent der Fälle eines von fünf Mal treffen. Scheint uns das nicht etwas plausibler? Naja, zumindest ist es wahrscheinlicher, als wenn wir eine Trefferchance von 50/50 annehmen.

Definieren wir nun ein Modell, dass 100 Variationen der trefferchance (d.h. 100 Paramterwerte) beinhaltet. Außerdem schließen wir von vornherein aus, dass die Trefferchance größer ist als 30 %.

set.seed(42)

USB <-
  tibble(c_Treffer = seq(0, 1, length.out = 100),
         Priori = case_when(c_Treffer > 0.3 ~ 0,
                            c_Treffer <= 0.3 ~ 1))%>%
  mutate(Likelihood = dbinom(1, size = 5, prob = c_Treffer),
         unst_Post = Priori * Likelihood,
         Post = unst_Post / sum(unst_Post))

Das Modell steht. Wir sehen, dass Beobachtungen mit einer Trefferchance von mehr als 30 % in der Post-Spalte eine Wahrscheinlichkeit von Null haben. Außerdem haben Beobachtungen mit einer Trefferchance von ca. 20 die höchste Wahrscheinlichkeit.

Wie groß ist die Wahrscheinlichkeit, dass die echte Trefferchance (laut unserem Modell) zwischen 12,5 % und 27,5 % liegt?

USB%>%
  filter(c_Treffer > 0.125 & c_Treffer < 0.275)%>%
  summarise(sum(Post))
